# Sistema de Gestão Hoteleira

## Descrição

O Sistema de Gestão Hoteleira é uma aplicação web desenvolvida como Projeto de Fim de Curso, com o objetivo de automatizar e facilitar a gestão de hotéis. O sistema permite o controlo de clientes, quartos, reservas e pagamentos, proporcionando maior eficiência na administração hoteleira.

## Funcionalidades

* Cadastro e gestão de clientes;
* Cadastro e gestão de quartos;
* Registo de reservas;
* Check-in e Check-out de hóspedes;
* Gestão de pagamentos;
* Consulta de disponibilidade de quartos;
* Painel administrativo;
* Relatórios e consultas de informações.

## Instalação

1. Clone o repositório:

```bash
git clone https://github.com/BeloneAlberto-dev/projecto-fim-curso-gestao-hoteleira.git
```

2. Copie a pasta do projeto para:

```text
C:\xampp\htdocs\
```

3. Inicie o Apache e o MySQL no XAMPP.

4. Crie uma base de dados no phpMyAdmin.

5. Importe o ficheiro SQL localizado na pasta do projeto.

6. Configure os dados de conexão com a base de dados.

7. Acesse o sistema através do navegador:

```text
http://localhost/hotel
```
